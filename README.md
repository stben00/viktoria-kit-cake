# Viktoria Kit Cake — Premium Website Demo

Premium single-page website for a couture cake atelier in Warsaw.

## Files
- `index.html` — final cake studio website
- `pitch.html` — sales/portfolio presentation page for showing the project to potential clients
- `img/` — project images

## What this demonstrates
- premium visual identity
- responsive HTML/CSS/JS site without frameworks
- multilingual UI (PL / UA / EN)
- real product gallery and founder section
- WhatsApp / Instagram conversion flow
- price calculator
- Google Maps embed
- reviews with photo cards
- smooth animations, loader, scroll progress and 3D hero effect

## Live usage
Upload the repository to GitHub Pages, Netlify, Vercel, or any static hosting.

Recommended demo links:
- `/index.html` — client-facing website
- `/pitch.html` — sales page to show the website as a portfolio case
