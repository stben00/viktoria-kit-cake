# Viktoria Kit Cake — Premium Website Demo

Premium single-page website for a couture cake atelier in Warsaw.

## Pages
- `index.html` — client-facing premium cake studio website
- `pitch.html` — sales/portfolio presentation page

## Features
- responsive HTML/CSS/JS
- multilingual UI (PL / UA / EN)
- real product gallery and founder section
- WhatsApp / Instagram conversion flow
- price calculator
- Google Maps embed
- reviews with photo cards
- smooth animations, loader, scroll progress and 3D hero effect
